카레 만들기

문제

ACM-ICPC 한국 리저널이 가까워지자, 집중 연습이 필요하다고 생각한 당신은 친구 집에서 열리는 경쟁 프로그래밍 캠프에 참가하기로 했습니다. 참가자들의 주장에 따라, 식사는 각자가 요리하기로 결정했습니다.

캠프 첫날 밤, 참가자들은 연습을 마치고 저녁 식사를 준비하기 시작했습니다. 경쟁 프로그래밍뿐만 아니라 요리에서도 프로라는 별명을 갖고 있는 당신은, 눈 깜짝 할 사이에 자신의 요리를 끝내고 많은 시간이 남았습니다. 그래서 당신은 다른 사람이 카레를 만드는 것을 도와주기로 했습니다.

W0리터(L)의 물과 R0그램(g)의 카레가루를 섞은 카레가 있습니다. 카레가루는 R(g)씩 포장되어 있으며, 충분히 많다고 가정합시다. 당신은 농도 C(g/L)인 카레가 가장 맛있다고 생각하기 때문에, 이 카레에 물과 카레가루를 적당히 더 넣어 농도를 C(g/L)로 만들려고 합니다. 여기서, 카레가루 R0(g)를 물 W0(L)에 넣은 카레의 농도는 R0 / W0(g/L)이며, 여기에 R(g)의 카레가루를 X개 넣고 물을 Y(L) 더 넣으면, 농도는 (R0 + X R) / (W0 + Y)(g/L)이 됩니다. 카레가루를 너무 많이 사용하는 것은 피하고 싶어서, 가능한 한 적은 개수의 카레가루를 첨가하려고 합니다.

농도 R0/W0(g/L)인 카레에 카레가루, 물, 또는 둘 다를 적당히 넣어 농도 C(g/L)인 카레를 만들 때, 첨가해야 하는 카레가루의 개수 X의 최솟값을 구하세요.

다음 사항에 주의하세요.

  • 추가할 카레가루의 개수 X는 0 이상의 정수여야 합니다. 예를 들어 1/3개의 카레가루를 넣을 수 없습니다.
  • 추가할 물의 양 Y는 0 이상의 실수입니다. 꼭 정수일 필요는 없습니다.
  • 카레가루나 물, 또는 둘 다를 추가하지 않고 농도 C의 카레를 만들 수 있을 수도 있습니다.
  • 카레가루와 물은 농도 C의 카레를 만들기에 충분하다고 가정합니다.

입력

입력은 여러 개의 테스트 케이스로 구성되어 있습니다. 각 테스트 케이스는 다음과 같은 형식으로 주어집니다.

R0 W0 C R

여기서, R0, W0, C, R은 각각 지금의 카레에 들어 있는 카레가루의 양(g)과 물의 부피(L), 만들고자 하는 카레의 농도(g/L), 카레가루 포장 하나의 양(g)을 나타냅니다. 이 값들은 모두 1 이상 100 이하의 정수입니다. 입력의 끝에는 네 개의 0이 주어집니다.

출력

각 테스트 케이스에 대해, 원하는 농도의 카레를 얻기 위해 필요한 카레가루 포장의 최소 개수를 한 줄에 출력합니다. 추가한 물의 양은 출력하지 않습니다.

입력 범위 내의 모든 입력에 대해서, 정답은 32비트의 부호 있는 정수 범위에 있음이 보장됩니다.
