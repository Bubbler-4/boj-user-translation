나누기

문제

길이 N의 정수 수열 A = (A1, A2, …, AN)가 주어진다. 수열 A의 값들은 모두 다르다.

수열을 최댓값을 기준으로 나누어서, 최댓값보다 앞에 있는 수들의 합과 최댓값보다 뒤에 있는 수들의 합을 출력하시오.

다시 말해, 수열 A의 최댓값이 Ax라고 하면, A1 + A2 + … + Ax-1와 Ax+1 + Ax+2 + … +AN를 출력하시오.

최댓값 앞에 값이 없으면, 최댓값 앞에 있는 값의 합은 0이다.

마찬가지로, 최댓값 뒤에 값이 없으면, 최댓값 뒤에 있는 값의 합은 0이다.

입력

입력은 다음과 같은 형식으로 표준 입력을 통해 주어진다.

N
A1 A2 … AN

출력

첫 번째 줄에는 수열 A의 최댓값보다 앞에 있는 수들의 합을 출력한다.

두 번째 줄에는 수열 A의 최댓값보다 뒤에 있는 수들의 합을 출력한다.

제한

  • 1 ≤ N ≤ 100
  • 1 ≤ Ai ≤ 2000 (1 ≤ i ≤ N)
  • Ai ≠ Aj (1 ≤ i < j ≤ N)
  • 이 수열의 최댓값은 16이다. 16보다 앞에 있는 9와 3의 합은 12, 16보다 뒤에 있는 8과 1의 합은 9이다.
  • 이 수열의 최댓값은 121이다. 121 앞에는 수가 없으므로 0을 출력한다. 121의 뒤에 있는 8, 5, 4, 1, 3의 합은 21이다.
  • 최댓값의 어느 쪽에도 수가 없다.
