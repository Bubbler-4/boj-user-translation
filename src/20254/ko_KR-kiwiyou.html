사이트 스코어

문제

ICPC 월드 파이널행 티켓을 얻어내기 위해 다양한 대학에서 온 팀이 ICPC 리저널에서 경쟁한다. 각 리저널에 할당된 티켓의 수는 다를 수 있다. 아시아-태평양 하위 리저널에서는 사이트 스코어라는 점수에 기반해 티켓을 할당한다.

사이 스코어는 리저널 본선 혹은 예선에서 적어도 한 문제를 푼 팀과 대학만을 고료한다. 2020년 타이베이-신주 리저널의 사이트 스코어 계산 공식은 작년보다 간단하다.

  • 본선에서 적어도 한 문제를 푼 대학의 수를 UR이라 하자.
  • 본선에서 적어도 한 문제를 푼 팀의 수를 TR이라 하자.
  • 예선에서 적어도 한 문제를 푼 대학의 수를 UR이라 하자.
  • 예선에서 적어도 한 문제를 푼 팀의 수를 TR이라 하자.

2020 타이베이-신주 리저널의 사이트 스코어는 56UR + 24TR + 14UO + 6TO이다. 2020 타이베이-신주 리저널의 사이트 스코어를 계산하는 프로그램을 작성하자.

입력

첫째 줄에 양의 정수 UR, TR, UO, TO가 주어진다.

출력

2020 타이베이-신주 리저널의 사이트 스코어를 출력한다.

제한

  • 0 < UR ≤ TR ≤ 120
  • 0 < UO ≤ TO ≤ 1000

노트

지문에서 설명한 사이트 스코어 계산 공식은 실제와 다르다.
