크레파스 정리하기

문제

요코하마군은 케이스에 들어간 크레파스를 가지고 있다. 크레파스의 종류는 문자 하나로 표시되며, 영어 소문자・영어 대문자・숫자 중 하나이다. 케이스 안에는 같은 종류의 크레파스가 여러 개 들어있을 수도 있다.

어느날, 케이스에 들어간 크레파스를 가족에게 빌려줬는데, 순서가 어질러진 상태로 돌려받았다. 크레파스를 바르게 정렬하는 것을 좋아하는 요코하마군은, 다음과 같은 순서로 크레파스를 정리하기로 했다.

  • 영어 소문자는 영어 대문자보다 왼쪽에 정리되고, 영어 대문자는 숫자보다 왼쪽에 정리된다.
  • 영어 소문자는 알파벳 순으로 빠른 문자가 왼쪽에 정리된다.
  • 영어 대문자는 알파벳 순으로 빠른 문자가 왼쪽에 정리된다.
  • 숫자는 숫자가 작은 순으로 왼쪽에 정리된다.

정리가 끝난 뒤 크레파스의 순서를 구하라.

입력

입력은 여러 데이터 세트로 구성된다. 각 데이터 세트는 다음과 같은 형식으로 표현된다.

S

각 데이터 세트는 크레파스의 순서를 나타낸 문자열 S 의 행 하나로 구성된다. S 의 왼쪽에서부터 i 번째의 문자는 왼쪽에서부터 i 번째에 들어있는 크레파스의 종류를 나타낸다.

S 길이는 1 이상 1,000 이하이며, S 는 영어 소문자・영어 대문자・숫자로 구성된 문자열이다.

입력의 마지막은 # 만 있는 행으로 구성된다. 데이터 세트는 50 개 이내이다.

출력

각 데이터 세트에 대해서, 정리가 끝난 뒤의 크레파스의 순서를 나타내는 문자열을 한 줄로 출력하라.
