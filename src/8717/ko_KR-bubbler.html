띠

문제

한별이는 몇 개의 정수가 일렬로 적혀 있는 띠를 발견했다. 한별이는 이 띠를 한 곳에서 잘라, 왼쪽 띠에 적혀 있는 정수의 합과 오른쪽 띠에 적혀 있는 정수의 합의 차이를 최소로 하고 싶다. 양쪽 띠에는 정수가 적어도 하나 이상 있어야 한다. 이 차이의 최솟값을 구하시오.

입력

입력의 첫 번째 줄에는 정수의 개수 n (2 ≤ n ≤ 106)이 주어진다. 두 번째 줄에는 n개의 정수 ai (-103 ≤ ai ≤ 103)가 띠에 쓰여져 있는 순서대로 주어진다.

출력

두 조각에 쓰여 있는 정수들의 합의 차이의 최솟값을 한 줄에 출력한다.

힌트

4번째 수의 뒤에서 1, 2, 3, 4|5, 6과 같이 자르면 차이는 |(1 + 2 + 3 + 4) - (5 + 6)| = 1이 된다.
