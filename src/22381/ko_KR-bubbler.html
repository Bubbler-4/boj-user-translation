독의 늪

문제

당신은 레트로 롤플레잉 게임을 플레이하고 있다. 이 게임의 필드는 가로 100칸, 세로 100칸의 격자로 이루어져 있다. 이 필드의 왼쪽에서 x번째 세로줄, 위에서 y번째 가로줄에 있는 칸을 (x, y)로 쓴다. 당신이 조작하는 캐릭터는 이 필드에서 상하좌우로 한 번에 한 칸씩 이동할 수 있다.

이 캐릭터는 현재 (X0, Y0)에 위치해 있다. 당신은 이 캐릭터로 N개의 목적지를 순서대로 방문하려고 한다. 그런데 이 필드의 몇몇 칸은 독의 늪이기 때문에 움직일 때 주의가 필요하다. 캐릭터가 독의 늪으로 이동하면 캐릭터가 대미지를 받는다. 독의 늪이 아닌 칸은 독이 없는 땅으로, 캐릭터가 그 칸으로 이동하더라도 대미지를 받지 않는다. 대미지를 받는 여부는 캐릭터가 이동할 때 도착하는 칸에 의해서 결정된다. 예를 들어, 독의 늪에서 땅으로 이동할 때는 대미지를 받지 않는다.

이 필드에서 맨 왼쪽 위 칸이 (A, B)이고 맨 오른쪽 아래 칸이 (C, D)인 직사각형 영역은 땅이고, 나머지는 늪이다. 캐릭터가 독의 늪에서 대미지를 받는 횟수를 최소화하여 N개의 목적지를 순서대로 방문할 때, 캐릭터가 대미지를 받는 횟수를 구하시오.

입력

입력은 최대 50개의 테스트 케이스로 구성되어 있다. 각 테스트 케이스는 다음과 같은 형식으로 주어진다.

N
A B C D
X0 Y0
X1 Y1
X2 Y2
...
XN YN

테스트 케이스의 첫 줄에는 목적지의 개수 N이 주어진다. (1 ≤ N ≤ 100)

두 번째 줄에는 독이 없는 땅의 직사각형 범위를 나타내는 정수 A, B, C, D가 주어진다. 캐릭터가 이동하는 칸 (x, y)가 A ≤ x ≤ C, B ≤ y ≤ D를 만족하면 캐릭터가 대미지를 받지 않고, 나머지 경우에는 캐릭터가 대미지를 받는다. (1 ≤ A ≤ C ≤ 100, 1 ≤ B ≤ D ≤ 100)

세 번째 줄에는 캐릭터의 현재 위치 (X0, Y0)를 나타내는 두 정수 X0, Y0가 주어진다. 그 다음 N개의 줄에는 순서대로 방문할 목적지의 위치가 주어진다. 3 + i번째 줄에는 i번째 목적지의 위치 (Xi, Yi) 를 나타내는 두 정수 Xi와 Yi가 주어진다. (1 ≤ Xi, Yi ≤ 100, 모든 0 ≤ j < k ≤ N에 대해 (Xj, Yj) ≠ (Xk, Yk))

입력의 끝에는 0이 하나 주어진다.

출력

각각의 테스트 케이스에 대해, 캐릭터가 대미지를 받은 횟수를 한 줄에 출력한다.

힌트

예시 입력의 첫 번째 테스트 케이스의 상황은 아래 그림과 같다. (3, 3)에서 (5, 3)까지는 땅이지만, (6, 3)과 (7, 3)은 독의 늪이므로 첫 번째 목적지에 도착할 때까지 2번의 대미지를 받는다. 첫 번째 목적지에서 두 번째 목적지로 갈 때 4번 아래로 이동하는 것이 최단 거리이지만, 모든 칸이 독의 늪이기 때문에 4번 대미지를 받는다. 다시 땅으로 우회하여 그림과 같이 이동하면 (6, 3)，(6, 7)，(7, 7)에서 3번 대미지를 받는다. 따라서 받는 대미지의 최소 횟수는 5회이다.
