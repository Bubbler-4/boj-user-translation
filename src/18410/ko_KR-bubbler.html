머지

문제

길이 N의 정수 수열 A = (A1, A2, ..., AN)와, 길이 M의 정수 수열 B = (B1, B2, ..., BM)가 주어진다. 수열 A와 B는 각각 단조 증가 수열이다. 즉 A1 ≤ A2 ≤ … ≤ AN, B1 ≤ B2 ≤ … ≤ BM를 만족한다.

아래의 알고리즘을 사용하여 길이 N + M의 수열 C = (C1, C2, ..., CN+M)를 만든다.

  1. 처음에 C는 비어 있다.
  2. A와 B가 둘 다 비어 있으면 이 알고리즘이 종료된다.
  3. A와 B 중 한 쪽이 비어 있으면, 비어 있지 않은 쪽의 수열을 t로 둔다. 둘 다 비어있지 않으면, 둘 중 첫 번째 값이 작은 쪽의 수열을 t로 둔다. 두 값이 같다면, 수열 A를 t로 둔다.
  4. t의 첫 번째 값을 C의 끝에 추가한다.
  5. t의 첫 번째 값을 제거한다.
  6. 2번으로 돌아간다.

수열 A와 B가 주어졌을 때, 이 알고리즘에 의해 만들어지는 수열 C를 출력하는 프로그램을 작성하시오.

입력

입력은 아래의 형식으로 표준 입력을 통해 주어진다. 

N M
A1 A2 … AN
B1 B2 … BM

출력

표준 출력을 통해 N + M개의 줄에 걸쳐 답을 출력한다.

k (1 ≤ k ≤ N + M)번째 줄에는 Ck의 값을 출력한다.

제한

  • 1 ≤ N ≤ 500
  • 1 ≤ M ≤ 500
  • 1 ≤ A1 ≤ A2 ≤ … ≤ AN ≤ 2000
  • 1 ≤ B1 ≤ B2 ≤ … ≤ BM ≤ 2000

A = (1, 2), B = (2)이다. 아래의 과정에 의해 수열 C가 만들어진다.

  • A의 첫 번째 값은 1, B의 첫 번째 값은 2이므로, A의 첫 번째 값을 C에 추가하고 A에서 삭제한다.
  • A의 첫 번째 값은 2, B의 첫 번째 값은 2이므로, A의 첫 번째 값을 C에 추가하고 A에서 삭제한다.
  • A가 비어 있으므로, B의 첫 번째 값을 C에 추가하고 B에서 삭제한다.
  • A와 B가 모두 비어 있으므로, 알고리즘을 종료한다.

알고리즘이 종료되었을 때, C = (1, 2, 2)이다.
