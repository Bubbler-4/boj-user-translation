치즈

문제

올해도 JOI마을에서 치즈를 만들기 시작했고, 쥐가 둥지에서 나오기 시작했습니다. JOI마을은 격자 배치되어 있으며, 각 격자는 둥지, 치즈 공장, 장애물, 빈칸 중 하나입니다. 쥐는 둥지에서 나와 모든 치즈 공장을 방문하여 치즈를 먹습니다.

마을에는 $N$개의 치즈 공장이 있으며 모든 공장은 치즈를 하나만 생산합니다. 치즈의 경도는 공장마다 다르며, 마을에는 경도 $1$부터 $N$까지의 치즈를 생산하는 공장이 정확히 하나씩 존재합니다.

쥐는 처음에 힘 $1$을 갖고 있습니다. 치즈를 하나 먹으면 힘이 $1$ 증가하며, 힘보다 큰 값의 경도를 가진 치즈는 딱딱해서 먹을 수 없습니다.

쥐는 동서남북에 인접한 격자로 이동할 수 있으며, 한 번의 이동에는 1분이 소요됩니다. 다만 장애물이 있는 격자로는 갈 수 없습니다. 또한, 치즈 공장에서 치즈를 먹지 않고 지나칠 수 있습니다.

쥐가 모든 치즈를 먹는데 걸리는 최단 시간을 구해보자. 단, 쥐가 치즈를 먹는 데 걸리는 시간은 무시한다.

입력

$H+1$줄이 입력으로 주어진다.

첫 번째 줄에는 정수 $H, W, N$ $(1 \le H \le 1000，1 \le W \le 1000，1 \le N \le 9)$가 주어진다. $H$는 격자의 높이, $W$는 격자의 너비, $N$은 치즈 공장의 수를 의미한다.

이후 $H$개 줄에 격자의 상태가 주어진다. 각 줄은 $W$개의 문자로 구성되어 있으며S, 1, 2, ..., 9, X, .만 주어진다. 예를 들어 북쪽에서 $i (1 \le i \le H)$번째, 서쪽에서 $j(1 \le j \le W)$번째 격자는 입력의 $i+1$번째 줄 $j$번째 문자이다.

S는 둥지, X는 장애물, .는 빈칸, 1, 2, ..., 9는 해당 경도의 치즈를 생산하는 공장을 의미한다. 입력에 둥지와 $1, 2, \cdots, N$ 경도의 치즈를 생산하는 공장이 각각 하나씩 주어지며, 쥐가 모든 치즈를 먹을 수 있음이 보장된다.

출력

쥐가 모든 치즈를 먹기까지 걸리는 최단 시간(분)을 나타내는 정수를 출력하라. 치즈를 먹는 데 걸리는 시간은 무시한다.
