윤년

문제

일반적으로, 연도 x가 윤년인지는 다음과 같이 정의되어 있다.

  1. x가 400의 배수이면 윤년이다.
  2. x가 400의 배수가 아닌 100의 배수이면 윤년이 아니다.
  3. x가 100의 배수가 아닌 4의 배수이면 윤년이다.
  4. 그 외의 경우에는 윤년이 아니다.

이것을 다음과 같이 일반화할 수 있다. 수열 A1, ..., An가 주어질 때, x년이 "일반화된 윤년"인지 여부를 다음과 같이 정의한다.

  1. x가 Ai의 배수인 가장 작은 i (1 ≤ i ≤ n)가 있을 때, i가 홀수이면 윤년이고, 짝수이면 윤년이 아니다.
  2. 이러한 i가 없는 경우에는, n이 홀수이면 윤년이 아니고, 짝수이면 윤년이다.

예를 들어, A = [400, 100, 4]에 의해 정의되는 일반화된 윤년은 보통의 윤년과 같다.

수열 A1, ..., An과 양의 정수 l, r이 주어진다. l ≤ x ≤ r을 만족하는 양의 정수 x에 대해, x년이 A에 대한 일반화된 윤년이 되는 x의 개수를 구하시오.

입력

입력은 최대 50개의 테스트 케이스로 구성되어 있다. 각 테스트 케이스는 다음과 같은 형식으로 주어진다.

n l r
A1
A2
…
An

정수 n은 1 ≤ n ≤ 50을 만족한다. l, r은 1 ≤ l ≤ r ≤ 4000을 만족한다. 각각의 i에 대해 정수 Ai는 1 ≤ Ai ≤ 4000을 만족한다.

입력의 끝에는 3개의 0이 주어진다.

출력

각 테스트 케이스에 대해, 정답을 한 줄에 출력한다.
