박테리아의 증식 (Large)

문제

미생물 연구원인 파스칼은 특이한 증식 경향을 보이는 박테리아를 발견하였다. 현재 x마리의 박테리아가 있다면, 이상적인 환경에서 1시간 후에는 xx마리로 증식한다.

이 특성을 더 깊이 조사하기 위해, 3개의 정수 A, B, C가 주어지면 A마리의 박테리아가 B 시간 후에 몇 마리가 될지 계산하는 프로그램을 작성하시오. 정답의 값이 매우 클 수 있으므로, 정답을 C로 나눈 나머지를 대신 출력한다.

입력

첫 줄에는 테스트 케이스의 개수 T가 주어진다. 그 다음 줄부터 T개의 테스트 케이스가 주어진다.

각 테스트 케이스에 대해, 3개의 정수 A, B, C가 공백으로 구분되어 한 줄에 주어진다.

제한

  • 1 ≤ T ≤ 500
  • 1 ≤ A ≤ 1000
  • 1 ≤ C ≤ 1000
  • 1 ≤ B ≤ 1000

출력

각 테스트 케이스에 대해

Case #X: N

와 같이 한 줄에 출력한다. 여기서 X는 1부터 시작하는 테스트 케이스의 번호, N은 박테리아의 수를 C로 나눈 나머지이다.
