용산에 간 승억이

문제

쓰고 있는 컴퓨터가 너무 느려져서 답답해진 승억이는 새 컴퓨터를 사기 위해 용산에 갔다. 승억이는 이왕 용산까지 온 거 여기서 살 수 있는 가장 최고의 컴퓨터를 사서 돌아가겠다고 다짐했다.

어떻게 하면 최고의 컴퓨터를 고를 수 있을까 고민하던 승억이는 몇 년간의 분석 끝에 엄청난 사실을 알아냈다. 컴퓨터의 램 용량이 $R$ GB, CPU 클럭 수가 $S$ MHz이고 저장공간 용량이 $D$ GB라고 할 때, $2 \times R + 3 \times S + D$ 의 값이 큰 컴퓨터가 사양이 더 좋다는 것이다!

암산을 잘 하지 못하는 승억이는 여러분의 도움이 필요하다. 새 컴퓨터를 장만하고 싶은 승억이를 위해 여러 컴퓨터의 램, CPU, 저장 공간 정보가 주어졌을 때 가장 좋은 컴퓨터와 두 번째로 좋은 컴퓨터의 이름을 출력하는 프로그램을 만들어주자!

입력

첫 번째 줄에 컴퓨터의 개수 $N$ ($0$ ≤ $N$ ≤ $10,000$) 이 주어진다.

두 번째 줄부터 $N$개의 줄에 걸쳐 컴퓨터의 사양이 주어진다. $i+1$번째 줄에는 컴퓨터의 이름 $C_i$, 램 용량 $R_i$, CPU 클럭 수 $S_i$, 저장 공간의 크기 $D_i$가 주어진다. ($1$ ≤ $R_i$ ≤ $128$, $1$ ≤ $S_i$ ≤ $4,000$, $1$ ≤ $D_i$ ≤ $3,000$)

이때 $C_i$은 길이가 20보다 작은 문자열이며, 각 줄의 컴퓨터의 이름, RAM, CPU 클럭과 저장 공간 사이에는 공백이 하나씩 주어진다.

출력

첫 번째 줄에 가장 성능이 좋은 컴퓨터의 이름을, 두 번째 줄에 두 번째로 사양이 좋은 컴퓨터의 이름을 출력한다.

만약 두 컴퓨터의 사양이 같을 경우, 사전순으로 앞서는 컴퓨터가 사양이 더 좋다고 가정한다. 예를 들어 Apple의 컴퓨터는 Dell의 컴퓨터보다 사양이 좋다. 컴퓨터가 한 대밖에 없을 경우 그 컴퓨터의 이름을 한 번만 출력하면 된다.

승억이의 계산식에 따르면 ABC, DEF, GHI, JKL 컴퓨터의 사양은 각각 93, 110, 30, 120이다. 따라서 JKL 컴퓨터가 가장 사양이 좋고 DEF 컴퓨터가 두 번째로 사양이 좋다.
